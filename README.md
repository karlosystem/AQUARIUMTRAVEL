# Aquarium Travel

Sitio web oficial de la agencia de viajes **Aquarium Travel**, diseñado para la gestión de paquetes turísticos, ofertas de viajes y visualización de destinos.

## 📋 Descripción del Proyecto
Aquarium Travel es una plataforma desarrollada íntegramente en PHP (versión 5.6), sin el uso de frameworks modernos. El sitio permite la gestión de ofertas de viajes, visualización de destinos, paquetes turísticos, reservas y cuenta con un panel de control administrativo para la gestión de contenidos.

## 🚀 Tecnologías Utilizadas
* **Backend:** PHP 5.6
* **Base de Datos:** MySQL
* **Frontend:** HTML5, CSS3, JavaScript
* **Servidor Local:** Apache (vía XAMPP)
